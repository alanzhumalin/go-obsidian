### **Строки в Go**

#### **Что под капотом?**

В Go строка (`string`) представляет собой неизменяемую последовательность байт, доступную только для чтения.

Структура строки:
```Go
type string struct {
    ptr *byte // Указатель на массив байт
    len int   // Длина строки
}
```

- Строка хранит указатель на байты и их длину.
- Строки неизменяемы: любая операция изменения создаёт новую строку.

Пример:
```Go
s := "hello"
fmt.Println(len(s))       // 5
fmt.Println(s[0])         // 104 (байт 'h')
fmt.Println(s + " world") // Создаёт новую строку "hello world"
```

Основные св-ва:
- Неизменность: После создания строки её содержимое нельзя изменять. Любая операция, которая кажется изменяющей строку (например, добавление символа), создаёт новую строку
- Кодировка: Строки в Go используют UTF-8, что делает их удобными для работы с Unicode-символами

Пример:
```Go
s := "Привет, ковбой!"
fmt.Println(s) // Выведет: Привет, ковбой!
```

Соединение нескольких строк в одну целую выполняется с помощью оператора сложения `+`:
```Go
var variable1 = "hello"
var variable2 = "world" 
var space = " " 
var variable3 = variable1 + space + variable2 
fmt.Println(variable3) // ВЫВОД: hello world 
fmt.Println(variable2 + ", " + variable1) // ВЫВОД: world, hello
```

Складывать строки с другими типами нельзя
```Go
fmt.Println("i am " + 24 + " years old") // ОШИБКА
```

Также можно превратить последовательность явно указанных байтов в полноценную строку и работать уже с ней:
```Go
// последовательность байтов 
any_bytes := []byte{0x47, 0x65, 0x65, 0x6b, 0x73} 
// создаем строку 
any_string := string(myslice1) 
fmt.Println(any_string) // ВЫВОД: Geeks
```

Длина строки в Go соответствует количеству используемых байтов, а не числу символов. Это связано с тем, что некоторые символы занимают 2 и более байта:
```Go
length := len("hello") 
fmt.Println(length) // ВЫВОД: 5

length := len("привет") // указано 6 символов
fmt.Println(length) // ВЫВОД: 12
```

---

Взаимодействие с байтами и рунами:
- Байты: Строка - это последовательность байтов. Вы можете обратиться к строки как к слайсу байтов
```Go
// последовательность байтов 
any_bytes := []byte{0x47, 0x65, 0x65, 0x6b, 0x73} 
// создаем строку 
any_string := string(myslice1) 
fmt.Println(any_string) // ВЫВОД: Geeks

s := "Hello"
b := []byte(s)
fmt.Println(b) // Вывод: [72 101 108 108 111] (ASCII-коды символов)

```
- Руны: Для работы с Unicode-символами (включая символы, которые занимают более 1 байта), используют тип `rune`

Пример работы с байтами:
```Go
s := "Привет"
fmt.Println(len(s)) // Длина строки в байтах
fmt.Println(s[0])   // Первый байт
```
Пример работы с рунами:
```Go
s := "Привет"
for _, r := range s {
    fmt.Printf("%c ", r) // Выведет каждый символ строки
}
```

---
Изменение строки:
Хотя строка неизменяема, её можно преобразовать в слайс байтов или рун, изменить его, а затем обратно.

Пример:
```Go
s := "Привет"
b := []rune(s)  // Преобразуем в руны
b[0] = 'Д'      // Меняем первый символ
s = string(b)   // Преобразуем обратно в строку
fmt.Println(s)  // Выведет: Дривет
```
---
Сравнение строк:
- Строки сравниваются лексикографически (побайтно)
- Сравнение можно выполнять с помощью операторов `==`, `<`, `>`
Пример:
```Go
fmt.Println("a" < "b") // true
fmt.Println("кот" > "код") // true
```

Для того чтобы узнать длину строки, достаточно перевести строку в слайс рун:
```Go
fmt.Println(len([]rune("Привет, ковбой!"))) // Вывод: 15
```

### **Чем это отличается от работы с байтами:**

- Слайс рун работает с **символами**, а не с их байтовым представлением.
- Если бы ты использовал слайс байтов, пришлось бы вручную обрабатывать Unicode-символы, которые занимают несколько байтов.
