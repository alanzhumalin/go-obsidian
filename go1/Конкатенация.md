# Конкатенация строк в Go

Конкатенация — это объединение (склеивание) двух или более строк в одну строку.

## 1) Через + (самый простой способ)

```
package main

import "fmt"

func main() {
	first := "Alan"
	last := "Walker"

	full := first + " " + last
	fmt.Println(full) // Alan Walker
}

```

## 2) Через +=

```
s := "Hello"
s += ", "
s += "world"
fmt.Println(s) // Hello, world

```

## 3) Через fmt.Sprintf (когда нужен шаблон)

```
name := "Alan"
age := 21

s := fmt.Sprintf("Name: %s, Age: %d", name, age)
fmt.Println(s) // Name: Alan, Age: 21

```

## 4) Для многих склеек — strings.Builder (эффективнее)

```
package main

import (
	"fmt"
	"strings"
)

func main() {
	var b strings.Builder
	b.WriteString("Go")
	b.WriteString(" ")
	b.WriteString("lang")

	s := b.String()
	fmt.Println(s) // Go lang
}

```

## 5) Склейка среза строк — [strings.Join](app://-/index.html#)

```
parts := []string{"Go", "is", "fast"}
s := strings.Join(parts, " ")
fmt.Println(s) // Go is fast

```

## Что использовать

- 2–3 простые части: +
- Шаблон с числами/типами: fmt.Sprintf
- Много склеек в цикле: strings.Builder
- Есть []string: [strings.Join](app://-/index.html#)
