# Указатели

**Указатель — это переменная, которая содержит адрес другой переменной в памяти.**

##### **Пример:**

```Go
var x int = 10 
var ptr *int = &x // Указатель на x 
fmt.Println(*ptr) // 10 (разыменование указателя)
```

#### **Зачем нужны указатели?**

1. **Передача по ссылке:**
    - Изменение значения переменной в функции.

```Go
func update(val *int) {
    *val = 20
}

func main() {
    x := 10
    update(&x)
    fmt.Println(x) // 20
}
```
2. **Экономия памяти:**
	- Передача указателя вместо копии большого объекта.

#### **Какой с них прок?**

- **Оптимизация ресурсов:** Экономия памяти и времени.
- **Гибкость:** Указатели позволяют изменять данные вне текущей области видимости.
